Now to learn about how AI actually works. We will specifically look at a neural network.

A machine learning model needs to train off of data to "learn" so that it can make predictions. The information it trains on is called the "training data" and the information it tests its knowledge on is called the "test data." It's like studying for a history "test" by taking a practice or "training" test.
A neural network will first randomly guess, but it will use the training data to see where it is making a mistake and tweat its own "thinking" to do better next time. Just like we should all strive to do. :) This is what we call training the model.
The model specifically is made up of layers of neurons, which are just values from 0 to 1. The input data has multiple values ranging from 0 to 1 which can represent anything (say, pixels of an image) and then the network of neurons mathematically manipulates that input to reach an output which is a value(s) of 0 to 1.
